# Modern Responsive Multi-Section Website

A modern and fully responsive multi-section website built using HTML & CSS, designed with a clean UI and smooth user experience.

## 🚀 Features

* Responsive design for all devices (Mobile, Tablet, Desktop)
* Clean and modern UI/UX
* Multi-section layout (Articles, Gallery, Services, Team, Testimonials, Pricing, etc.)
* Well-structured and maintainable code
* استخدام Font Awesome للأيقونات و Google Fonts لتحسين التصميم

## 🛠️ Technologies Used

* HTML5
* CSS3

## 📌 Overview

This project demonstrates front-end development skills by building a complete website layout from scratch, focusing on responsiveness, design consistency, and code organization.
